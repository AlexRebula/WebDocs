WebDocs is a modern, responsive web theme created for web documentations. It is ready for you to simply start writing any kind of documentation.

 View on GitHub  Download .zip  Download .tar.gz

There are two versions of WebDocs available. They look and feel the same, however there if you look at the source code you will see that one is powered by KnockoutJS and the other with AngularJS.

Choose the one that fits you the most.
